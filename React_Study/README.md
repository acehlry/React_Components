## 개발환경

```sh
# 프로젝트 환경설정 - vite
npm install vite@latest

# React 중앙집중식 상태관리 라이브러리
npm install recoil

# API 통신 라이브러리
npm install axios

# CSS 스타일링 SASS/SCSS
npm install -D sass

# React Router
npm install react-router-dom localforage match-sorter sort-by

# Typescript에서 NodeJS 모듈 사용
npm install @types/node

# React Toast Popup
npm install react-simple-toasts
```
